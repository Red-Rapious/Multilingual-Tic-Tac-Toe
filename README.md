## Available Languages
- [Python](src/python)

## Running instructions
### Python
***Requirements:** Python 3.10 or higher (use of the `match` statement)*

Launch the following command at the root of the project:
```console
$ python3 src/python/main.py
```

For Windows:
```console
> py src/python/main.py
```
